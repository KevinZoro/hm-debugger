百度分析chrome插件

Background:
    百度分析chrome插件，用于分析百度搜索结果的点击情况，以及用户的搜索行为。
    上传这个的起因是公司的旧项目，在排查问题时，发现百度分析提供了chrome的插件，原地址在https://tongji.baidu.com/static/tongji/hm-debugger/#using，但看起来该插件已经停止维护了，直接下载安装会报错，提示"无法安装"
    为了能够继续使用该插件，我使用AI进行了修复，修复后可以成功安装了。有需要的朋友可以直接克隆这个git仓库，然后在chrome的扩展程序页面，打开"开发者模式"，点击"加载已解压的扩展程序"，选择克隆的仓库目录，即可安装修复后的插件。
